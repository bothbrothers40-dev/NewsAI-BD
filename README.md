# NewsAI-BD
MY first project

This repository contains a static HTML site for NewsAI BD. To publish the site:

1. Ensure an `index.html` file is present at the root (already done).
2. Enable **GitHub Pages** for this repository via Settings → Pages, selecting the `main` branch and `/ (root)` or let the workflow deploy automatically.
3. A GitHub Actions workflow (`.github/workflows/pages.yml`) is included which will deploy the `main` branch on every push.

Once enabled, the site will be available at `https://<your‑username>.github.io/NewsAI-BD/` or the custom domain you configure.
